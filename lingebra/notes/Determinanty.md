# Determinanty

$$det(A) = \sum_{\sigma \in S_n} sign(\sigma) \prod a_{i, \sigma(i)}$$


* $det(A) = det(A^T)$
* $det(A + e_i b^T) = det(A) + det(A + e_i(b^T - A_{*,i}$
* $det(A + e_i c) = c \cdot det(A)$
* $det(AB) = det(BA) = det(A) det(B)$
* $det(A^{-1}) = \frac{1}{det(A)}$
* Pokud je matice $A$ regulární tak $det(A) \not = 0$
* Pokud jsou dva řádky matice $A$ stejné $det(A) = 0$
